  • Tank Bot

    • A bot made for my Discord server using Discord.js and API's! The bot has a prefix "t."

      Command List:
        Info:
      • about - learn about Tank Bot
      • help - list of all commands
      • ping - a way to test Tank Bot's connection to Discord
      • pig - better way to test Tank Bot's connection to Discord
        Moderation:
      • ban - ban a member from this server
      • kick - kick a member from this server
      • ot - a way to tell someone they are off topic
        Fun:
      • 8ball - answers all questions with an 8ball response
      • meme - random Reddit meme
      • rate - get a rating from Tank Bot out of 10
      • roast - get roasted by Tank Bot
        Miscellaneous:
      • oh - a count of the times FrankzeeTank says oh
      • quote - get a random FrankzeeTank quote
      • tweet - tell Biden what to tweet
