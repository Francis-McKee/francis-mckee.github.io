Calculator GetKey

A TI-Basic calculator programming tool to get the numerical value of any key

Description:

This is a downloadable file for TI- Calculators that will help you get the numerical value for any key! This tool is for TI-Basic calculator programmers that are trying to handle user input.

Download and Install Instructions:

This program should work on any TI-83 calculators and newer, including the TI-83+, TI-84, TI-84+CE, TI-Nspire, and more! In order to download and use this tool on your calculator, you must have a cable to connect your calculator to your PC (most come with the calculator), as well as software on your computer that you use to send files to your calculator. You can download such software here.
